# 0-Degrees 
Stanley Milgram popularized the theory that everyone is connected within Six Degrees of Separation. We aim to bring that number down to 0. Despite this small world we live in, we can still feel disconnected. Our goal is to connect people who would not otherwise cross paths or interact together, to build lasting friendships and broaden their social networks by hosting fun interactive events and creating a welcoming environment for ALL students. Feel free to join our discord server, where nearly all of the actual communication is done: https://discord.gg/JnmUhS8X59 (UNC-CH students only!)

This repository currently contains an automated poll script to fetch one or more public alert RSS feeds (like Alert Carolina) on a schedule and posts new entries to Discord as native-looking embeds, styled to look like it's posted directly by that source, complete with its own name, icon, and color-coded severity. It remembers which alerts it has already sent by comparing it against the last-seen alert ID saved from the previous run so nothing gets duplicated, and posts only truly new entries to that source's Discord webhook--skipping routine "Normal conditions" posts. It runs entirely on GitHub's own infrastructure on a timer and is built such that adding another alert source in the future just means pointing it at a new feed rather than rewriting any of the underlying logic.
